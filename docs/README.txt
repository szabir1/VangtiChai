VangtiChai - CSE 489 Assignment 1

App Name:
VangtiChai

Description:
VangtiChai is an Android application that allows users to enter a money amount using a custom numeric keypad. The app calculates the required number of Taka notes for 500, 100, 50, 20, 10, 5, 2, and 1 taka denominations.

Implemented Features:
- Custom numeric keypad implemented from scratch
- Taka note calculation for 500, 100, 50, 20, 10, 5, 2, and 1
- Clear button to reset the amount
- Portrait and landscape layout support
- State preservation during screen rotation
- Layout values managed through sizes.xml
- Custom colors and button styling

Tested Device:
- realme RMX3471 physical Android device
- pixel xl
- nexus 10 tablet
- pixel 4 xl

Tested Orientations:
- Portrait mode
- Landscape mode
